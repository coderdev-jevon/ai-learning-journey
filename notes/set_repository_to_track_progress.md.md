## Step 1
	Make a folder and go to the directory in CMD

## Step 2
	Activate Git in the folder
```
git init
```

## Step 3
	Create a good folder structure and create your first file
```
AI-Learning-Journey/
│
├── daily-log/
├── projects/
├── notes/
└── README.md
```

## Step 4
	Your first commit
```
git add .
git commit -m "message/topic"
```

## Step 5
	Create a repo in GitHub
	Make sure it is public
	Don't tick README and Create

## Step 6
	Connect local repo to github
```
## first copy the github url of your repo
git remote add origin url
git branch -M main
git push -u origin main
## login first with username and token from github
	## tips for token, make classic and tick repo, copy the token and paste in the password later
```

## Step 7
	Update everyday and that's itoke
```
git add .
git commit -m "Day 2 - .."
git push
```
